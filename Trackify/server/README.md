# Expenses_Tracker_Backend

---

### Built With

* [Node.js](https://nodejs.org/en/)
* [Mongo DB](https://www.mongodb.com/)
* [Express.js](https://expressjs.com/)

---

### To run the server :
1. Clone the repo
2. Install for the node modules `npm install`
3. run the front end `npm start`
